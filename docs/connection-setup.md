# Подключение Trading Platform к торговым серверам

## Как подключить Trading Platform к MT4/MT5?

Для подключения Trading Platform к торговому серверу MetaTrader 4 или MetaTrader 5 необходимо:

1. Обратиться к команде Customer Success или в техническую поддержку Trading Platform
2. Предоставить данные доступа к Manager API торгового сервера (адрес сервера, логин и пароль Manager API)
3. Команда Trading Platform выполнит настройку подключения на своей стороне
4. После настройки данные с торгового сервера начнут поступать в систему

Подключение выполняется командой Trading Platform — установка дополнительного ПО со стороны брокера не требуется (за исключением плагинов для отдельных функций).

Подробные технические требования доступны по запросу у Customer Success.

## Как подключить Trading Platform к cTrader?

Trading Platform поддерживает интеграцию с платформой cTrader. Процесс подключения аналогичен MT4/MT5:

1. Обратиться к команде Customer Success
2. Предоставить необходимые данные доступа к cTrader Open API
3. Команда Trading Platform выполнит интеграцию

Для получения детальных технических требований по cTrader обратитесь в Customer Success.

## Какие API credentials нужны для подключения?

Для подключения торгового сервера к Trading Platform необходимы учётные данные Manager API:

- **MT4/MT5:** адрес торгового сервера, логин и пароль Manager API с необходимыми правами (чтение счетов, позиций, истории)
- **cTrader:** данные доступа к cTrader Open API

Точные требования к правам Manager API зависят от набора подключаемых функций. Обратитесь к Customer Success для получения актуального списка необходимых прав.

## Сколько времени занимает подключение к торговому серверу?

Стандартное время подключения торгового сервера к Trading Platform составляет **1–2 рабочих дня** с момента предоставления всех необходимых данных доступа.

Сбор исторических данных после подключения может потребовать дополнительного времени в зависимости от объёма данных.

## Можно ли подключить несколько торговых серверов одновременно?

Да, Trading Platform поддерживает одновременное подключение нескольких торговых серверов (нод). Каждый сервер подключается как отдельная нода в системе. Это позволяет:

- Управлять несколькими MT4, MT5 и cTrader серверами из единого интерфейса
- Видеть агрегированные данные по всем серверам
- Настраивать триггеры и мониторинг для каждого сервера отдельно или для всех сразу

Для подключения дополнительных серверов обратитесь к Customer Success.

## Нужно ли устанавливать плагины на MT4/MT5 сервер?

Для базовой работы Trading Platform установка плагинов на торговый сервер **не требуется**. Однако для некоторых расширенных функций может потребоваться установка серверного плагина:

- **Net Dynamic Leverage** — требуется плагин для корректной работы триггера динамического плеча. Плагин активируется по запросу.
- **CID/IP Monitoring** — при необходимости доступен плагин для получения данных CID и IP напрямую с торговых серверов, что исключает необходимость парсинга логов. После установки плагина и перезапуска сервера сбор данных начинается в течение 24 часов.

Для получения файлов плагинов и инструкций по установке обратитесь к Customer Success.

## Что делать если соединение с торговым сервером прервалось?

Если произошёл обрыв соединения Trading Platform с торговым сервером:

1. **Срочно свяжитесь с технической поддержкой** — не ждите, обрыв соединения является критической ситуацией
2. **Ожидайте ответа в течение нескольких минут** — при отсутствии реакции позвоните по телефону
3. **Соберите информацию** — время обрыва, скриншоты ошибок (при наличии), имя сервера/ноды
4. **Сообщите клиентам** (при необходимости), что проблема уже передана ответственной команде и решение в работе

Команда технической поддержки имеет SLA на реагирование по критическим инцидентам.
