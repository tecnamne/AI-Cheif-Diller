# Системные требования Trading Platform

## Общие требования

Trading Platform — это облачная SaaS-платформа. Для работы с ней не требуется установка серверного программного обеспечения на стороне брокера. Вся инфраструктура размещается и обслуживается командой Trading Platform.

## Требования для доступа пользователей

- **Браузер:** Google Chrome (рекомендуется), Mozilla Firefox, Microsoft Edge — актуальные версии
- **Интернет-соединение:** стабильное соединение с доступом к серверам Trading Platform
- **Разрешение экрана:** рекомендуется 1920x1080 и выше для оптимального отображения дашбордов

## Требования для подключения торговых серверов

Для интеграции торговых серверов с Trading Platform необходимо:

### MetaTrader 4 / MetaTrader 5
- Доступ к Manager API торгового сервера
- Открытый сетевой доступ (firewall) от серверов Trading Platform к торговому серверу по необходимым портам
- Manager API логин с правами на чтение данных (счета, позиции, история сделок)

### cTrader
- Доступ к cTrader Open API
- Соответствующие API-ключи

## Плагины (опционально)

Для расширенного функционала могут потребоваться плагины на торговом сервере MT4/MT5:
- **Net Dynamic Leverage** — серверный плагин для триггера динамического плеча
- **CID/IP Monitoring** — плагин для прямого сбора данных CID и IP без парсинга логов

Установка плагинов требует перезапуска торгового сервера. После установки данные начинают поступать в течение 24 часов.

## Безопасность

- Все соединения между Trading Platform и торговыми серверами зашифрованы
- Рекомендуется ограничить доступ к Manager API по IP-адресам серверов Trading Platform
- Доступ пользователей к платформе контролируется через систему ролей и разрешений
